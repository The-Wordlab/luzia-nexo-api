# Examples

Runnable Nexo integration examples.

Profile context note:
- Current stable profile field in webhook payloads is `profile.locale`.
- Additional consented profile fields are being expanded and documented progressively.
- Build examples and integrations to safely ignore unknown fields.

Folders:
- `webhook/minimal`
- `webhook/structured`
- `webhook/advanced`
- `partner-api/proactive`

Run all example tests from the repo root:

```bash
make test-examples
```
